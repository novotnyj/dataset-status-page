## Storing data

Use following input for storing a dataset info:

```
{
    "task": "store",
    "name": "actor-name",
    "dataset": "datasetId"
}
```

## Starting up a server with status page

Use following input for starting a status page server:

```
{
    "task": "serve",
    "intervals": ["day", "week", "two-weeks", "month", "two-months"]
}
```
